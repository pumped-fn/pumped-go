## Why Pumped Functions?

### ğŸ¯ **Problem: State Management Sucks**
Every state library makes you choose between:
- **Simple but Limited** (Context API) 
- **Powerful but Complex** (Redux)
- **Nice but Hard to Test** (Zustand)

### âœ¨ **Solution: What If It Was Just Functions?**
```typescript
// Your entire app state
const user = provide(() => ({ name: 'John' }));
const theme = provide(() => 'light');

// Update it
const setTheme = derive(theme.static, ctl => 
  (newTheme) => ctl.update(newTheme)
);

// React to changes
const greeting = derive(
  [user.reactive, theme.reactive],
  ([user, theme]) => `Hello ${user.name}! Theme: ${theme}`
);

// Test it
const scope = createScope(
  preset(user, { name: 'Test' }),
  preset(theme, 'dark')
);
```

**That's it. No providers. No reducers. No actions. Just functions.**
