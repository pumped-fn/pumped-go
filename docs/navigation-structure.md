## Mobile-Friendly Structure

For mobile navigation, group into fewer top-level items:

1. **Start** (Try, Install, Learn)
2. **Guide** (Concepts, Patterns, Examples)  
3. **Reference** (API, Types)
4. **Help** (FAQ, Discord)
